3.说明

1.根据switch表达式中的值，依次匹配各个case中的常量。一旦匹配成功，则进入case中，调用程序语句。

当调用完执行语句 如果没有break，则继续执行其他case中的语句，直到遇到break 或者流程结束。

2.break 使用在结构中，表示有一旦执行到此，就跳出结构。break根据实际情况是可选的 可有可无。

3.switch中只能有以下6种数据类型之一：byte、short、char、int、枚举类型、String类型（7.0以后） 。

​ boolean型是不可以的

4.case 后边一定放的是常量。不能放范围(要是可以的话 感觉和if-else没区别了)。

5.如果switch-case中执行语句相同 可以合并（例2）
